Team portrait reference images live here.

Use one image per team when you want generated rookie portraits to match a
team roster-photo style. For example:

```text
graphics/players/portrait_refs/MIN/media_day_base.png
graphics/players/portrait_refs/CHI/media_day_base.png
```

The generator uses these images for composition, lighting, background, crop,
and team-color direction only. Prompts tell the image model to create a new
fictional player from the prospect metadata and not copy the reference person's
face or identity.
